# Инструкция по сборке Alpaka MSG на GitHub

Этот репозиторий подготовлен для автоматической сборки Android APK с помощью GitHub Actions.

## Как собрать APK:

1. **Загрузите код на GitHub**:
   - Создайте новый репозиторий на вашем аккаунте GitHub.
   - Загрузите (push) содержимое этой папки в ваш репозиторий.

2. **Запустите сборку**:
   - Перейдите на вкладку **Actions** в вашем репозитории на GitHub.
   - В колонке слева выберите workflow: **Build Alpaka APK**.
   - Нажмите кнопку **Run workflow** (справа).
   - Дождитесь завершения процесса (обычно занимает 5-7 минут).

3. **Скачайте результат**:
   - Когда сборка завершится (появится зеленая галочка), нажмите на название запуска сборки.
   - Прокрутите страницу вниз до раздела **Artifacts**.
   - Нажмите на **alpaka-msg-apk**, чтобы скачать архив с готовым APK-файлом.

## Особенности этой сборки:
- Сервер по умолчанию: `alpakasoft.online`
- Название приложения: `Alpaka MSG`
- Сборка настроена как `release`, но без цифровой подписи (для простоты установки). При установке на телефон Android может предупредить о "неизвестном источнике", это нормально для неподписанных APK.

## Как изменить иконку перед сборкой:
1. Замените файл `assets/logo.png` на ваше изображение.
2. Запустите команду `flutter pub run flutter_launcher_icons` (если собираете локально) или просто обновите файлы в `android/app/src/main/res/mipmap-*`.
